Proposed Changes

Here is a list of changes we propose to the site, as well as the rationale behind these recommendations:

Convert to HTML5

The extra semantics of HTML5 will increase Search Engine Optimization (SEO) and improve site accessibility.

Convert to CSS3

CSS3 allows for more responsive code, without having to rely to heavily on plug-ins. This means better browser performance as the user does not need to download as many files to use the site.

CSS Reset

Employing a CSS reset will ensure a cleaner site, regardless of the browser being used. Even if the more advanced CSS3 features are not supported, the basic layout will be clear and clean, meaning the site will still be informative and useful.

Possible Future Developments

In the future, we would look to introduce the ability to support online quizzes, a page-specific sticky navigation to allow students to access areas of each page quicker, and minified files for even more enhanced browser performance.
